[Grav](http://getgrav.org) is a modern database-free CMS which stores Markdown content in a flexible and easy-to-access file structure. When paired with Grav's [Git Sync](https://github.com/trilbymedia/grav-plugin-git-sync) plugin it's a ideal future-friendly platform for open course and educational resource materials.
